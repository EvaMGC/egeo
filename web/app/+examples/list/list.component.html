What is...?

[data]: Array<<Model><T>>
Data is the array of our data that we show in the list . So, it is not all our data. You have to have all your data in a variable (allMyData) and filter it depending on the order, page, search... for calculate the input [data] (dataToShow = allMyData.filter(...)). The data must meet certain commitments. Look at our next example for more information
[fieldsMetaData] We need to know the fields of the data: label, type, if it has link, etc...
[order] How is the table ordered? By wich field? Asc or desc?
[hasDetail] It is a boolean that tell to the list if has to show the arrow at the right of each row. If the user click the row, we will have an event.
(openDetail) When the user click a row. You will receive the value of the id field correspondign.
[pagination] If you want to paginate the list
[totalElements] We need to know the fields of the data
(changePage) We need to know the fields of the data
(onOrderChange) We need to know the fields of the data
